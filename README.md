# 东方Project Wiki

## 项目简介

Touhou Project Wiki是一个基于 HTML 和 JavaScript 开发的桌面应用程序，用于查看东方project IP的作品和其相关的人物信息。该系统提供了完整的用户认证、信息跳转等功能，界面友好，操作简便。

## 功能特点

### 🎯 核心功能

- **用户管理**：支持用户注册、登录、密码修改

- **板块跳转**：可以根据需要查看不同板块的信息

- **电梯导航**：快速导航到需要查看的文段

- **信息隐藏**：仿萌娘百科的文段隐藏

### 🔐 安全特性

- 用户登录验证

- 密码修改需验证旧密码

- 会话管理

### 🎨 界面特色

- 统一的页面背景和毛玻璃效果

- 版权信息展示

- 友好的对话框和提示信息

- 页面背景的自动轮播

## 系统要求

### 最低配置

- **操作系统**：Windows 7/10/11

- **内存**：2GB RAM

- **存储空间**：600MB 可用空间

### 推荐配置

- **操作系统**：Windows 10/11

- **内存**：4GB RAM 或更高

- **存储空间**：800MB 可用空间

## 安装与运行

### 使用预编译版本

1. 下载 `Touhou Project Wiki.exe` 文件

2. 双击即可安装，无需安装其他依赖

### 首次使用

1. 启动程序后，使用默认账户登录：
   
   - **用户名**：Admin1
   
   - **密码**：123456

2. 或点击"注册"按钮创建新账户

### 主要操作流程

1. **浏览板块**：从顶部的导航栏可以查看不同的页面

2. **查看隐藏文段**：轻触文字蒙板可显示隐藏的信息

3. **文段解释**：轻触或点击绿色高亮文字可跳转或查看具体信息

## 技术栈
